📌 DESCRIÇÃO
Ambiente virtual navegável desenvolvido em Unity com suporte a Meta XR SDK.
Cena contém chão, skybox configurado, 5+ objetos 3D e movimentação testada em PC.

⚙️ CONFIGURAÇÃO TÉCNICA
- Unity 2022.3 LTS (ou superior)
- Meta XR SDK instalado via Package Manager
- XR Plugin Management: OpenXR ativado
- Build Settings: Plataforma Android (Meta Quest)

🎮 CONTROLES (TESTE PC)
- W/A/S/D ou Setas: Movimentação
- Mouse: Rotação de câmera (se configurado)
- O projeto foi testado e validado no Unity Editor antes do build Android.

📦 ORGANIZAÇÃO
- Assets/Scenes: MainScene.unity
- Assets/Scripts: PlayerMovement.cs, VRSceneSetup.cs
- Hierarquia organizada em pastas lógicas
- Nomenclatura padrão: [Tipo]_[Nome] (ex: CHAO_Principal, CUBE_Mesa01)

📚 APRENDIZADOS E DESAFIOS
- Configuração inicial do XR Plugin Management
- Diferença entre movimentação PC e controle VR
- Importância da organização de assets e nomenclatura
- Testes iterativos no Editor antes do deploy
